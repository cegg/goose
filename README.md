# goose
The Game of Goose Kata - Perl implementation, Ubuntu 16

to start:
$ ./bin/game.pl
(it's already executable)
or from bin:
./game.pl

to get help:
$ ./bin/game.pl --help

to get rules
$ ./bin/game.pl --rules

Prerequsite Perl Packages:

1. Getopt::Long
1. English
1. FindBin
1. File::Spec
1. Readonly
